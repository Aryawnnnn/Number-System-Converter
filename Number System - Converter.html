🔢 Number System Converter

Convert between Binary, Decimal, Hexadecimal, and Octal number systems

📚 Number System Information

Binary (Base 2) Uses digits: 0, 1
Example: 1010 = 10 in decimal
Octal (Base 8) Uses digits: 0-7
Example: 12 = 10 in decimal
Decimal (Base 10) Uses digits: 0-9
Example: 10 = standard counting
Hexadecimal (Base 16) Uses: 0-9, A-F
Example: A = 10 in decimal
